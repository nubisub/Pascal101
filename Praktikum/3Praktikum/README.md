1. Jalankan statement bahasa pemrograman Pascal, kemudian tuliskan output masing- masing statetement.
    ```pascal
    begin   
        writeln(1234:7);
        writeln(1234:6);
        writeln(1234:4);
        writeln(1234:1);
        writeln(1234:0);

        writeln('==========');

        writeln(12.35:10:4);
        writeln(12.35:10:3);
        writeln(12.35:8:1);
        writeln(12.35:8:0);
        writeln(-12.35:10:1);
        writeln(12.35:9);
        writeln(12.35:7);
        writeln(12.35:0:2);

        writeln('==========');

        writeln('a':6);
        writeln('abc':6);
        writeln('abc':9);

        readln;
    end.
    ```
